# 📊 Project 2 — Customer Interaction Tracker

A ready-to-use spreadsheet template for logging, tracking, and analyzing customer support interactions. Built to reflect real CRM workflows — the same logic I applied daily in Salesforce across 4.5 years of high-volume remote support.

---

## What's Included

| File | Description |
|------|-------------|
| [`customer-interaction-tracker.csv`](./customer-interaction-tracker.csv) | Main tracker with sample data — import into Excel or Google Sheets |
| [`how-to-use.md`](./how-to-use.md) | Step-by-step guide for using the tracker |

---

## Features

- **Interaction log** — date, time, contact method, customer details
- **Issue categorization** — tag by type for trend tracking
- **Resolution status** — Open, In Progress, Resolved, Escalated
- **Follow-up flags** — never let a case fall through the cracks
- **Resolution time tracking** — monitor your own performance
- **Notes field** — capture context that doesn't fit in dropdowns

---

## Why This Matters

CRM tools like Salesforce automate a lot of this — but understanding the *logic* behind the fields is what separates agents who just fill in boxes from agents who use documentation strategically. This tracker reflects that understanding.

---

## How To Use

1. Download `customer-interaction-tracker.csv`
2. Open in **Microsoft Excel** or **Google Sheets**
3. Each row = one customer interaction
4. See [`how-to-use.md`](./how-to-use.md) for field-by-field guidance
